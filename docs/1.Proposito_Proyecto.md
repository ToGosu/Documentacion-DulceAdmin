## 1. Propósito del Proyecto

El proyecto **DulceAdmin** tiene como propósito desarrollar un **sistema integral de facturación y gestión de inventario** diseñado específicamente para una panadería.  
El objetivo principal es **automatizar los procesos operativos** asociados a la venta, control de stock, generación de recibos y administración de productos, garantizando una gestión más eficiente y precisa del negocio.

El sistema busca **optimizar el flujo de trabajo**, reduciendo errores manuales, mejorando la trazabilidad de las operaciones y asegurando la disponibilidad de información en tiempo real.  
De esta manera, DulceAdmin permite al personal concentrarse en la atención al cliente y en las tareas productivas, mientras el software gestiona de forma confiable los registros de ventas, pagos y existencias.

A largo plazo, el proyecto pretende sentar las bases para una solución **escalable y adaptable**, que pueda evolucionar hacia un entorno distribuido o en la nube, sin perder la simplicidad y el enfoque local que demanda el negocio actual.
